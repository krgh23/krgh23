### 개발자 ‘조범희’를 소개합니다.

안녕하세요.

기술과 사용자 경험 사이의 가치를 유기적으로 연결하며, 실무 중심의 웹 서비스를 설계하고 개발하는 **개발자 조범희**입니다.

Spring Framework (Legacy)와 MyBatis를 활용해 DAO 구현 및 Mapper XML 작성, JSP 기반 블로그 시스템을 개발하며 **게시글 CRUD 기능**, **트리 구조의 댓글 등록 및 정렬 처리**, **Jsoup을 이용한 이미지 태그 파싱 및 첨부 파일 정보 추출** 기능 등을 구현한 경험이 있습니다.

또한 Spring Boot와 JPA를 기반으로 한 백엔드 시스템과 React 프론트엔드를 연동하여, (Spring Boot 프로젝트 기반의 REST API를 개발하였고, 이와 연동하기 위해 React.js 라이브러리를 활용하여 이커머스 SPA(Single Page Application) 프로그램을 구축했습니다. 세부적으로 Spring Security와 Spring Data JPA를 이용하여 REST API를 개발하였고, **포트원 API와의 결제 연동 시스템**까지 완성도 있게 구축했습니다.)

복잡한 비즈니스 로직을 구조적으로 설계하고 구현하는 데에서 큰 즐거움을 느끼며, 특히 백엔드와 프론트엔드를 유기적으로 연결하는 아키텍처 설계에 강점을 가지고 있습니다.
